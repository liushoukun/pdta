# 用户手册

## 📖 目录说明

面向最终用户的操作手册，帮助用户快速上手和使用系统。本目录包含按角色分类的操作指南、常见问题解答等内容。

## 📁 内容分类

- **快速入门** - 5分钟快速上手指南
- **登录注册** - 账号注册和登录说明
- **操作指南** - 按角色分类的详细操作步骤
  - 管理员操作
  - 运营人员操作
  - 客服操作
- **常见问题** - FAQ 和问题排查
- **视频教程** - 操作演示视频

- ✅ 使用大量截图和步骤说明
